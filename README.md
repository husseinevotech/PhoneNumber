## Project Description

A Single Page Application That Lists Phone Numbers
- Categorized By: 
    -   Country Name, 
    -   Country Code, 
    -   State(Valid or InValid),
    -   Number
- Filtered By:
    - Country Name
    - State
- Pagination.
